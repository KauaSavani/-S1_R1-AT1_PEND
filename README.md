# Rápido & Seguro Logística

Projeto de tela de login responsiva desenvolvido utilizando HTML5 e Bootstrap 5.

## Descrição

Este projeto consiste em uma interface de login para o sistema da empresa **Rápido & Seguro Logística**, com layout responsivo e estilização utilizando Bootstrap.

## Tecnologias Utilizadas

HTML5
Bootstrap 5
CSS (classes do Bootstrap)
JavaScript (Bootstrap Bundle)

## Divisão de Trabalho

### Kauã
Estruturação do HTML
Criação do formulário de login
Implementação da navbar e footer
Organização da semântica da página

### Enthony
Aplicação do Bootstrap 
Estilização
Responsividade (grid system)
Ajustes visuais: cards, botões, alinhamentos
Layouts

## Funcionalidades

Campo de e-mail
Campo de senha
Botão de login
Link "Esqueci minha senha"
Layout responsivo
Navbar adaptável para dispositivos móveis

## Responsividade

O projeto foi desenvolvido utilizando o sistema de grid do Bootstrap, como solicitado e adaptação  garantia para:

Desktop
Tablet
Smartphone
